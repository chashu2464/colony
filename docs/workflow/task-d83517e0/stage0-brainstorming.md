# M2.1 Stage 0: Brainstorming

## 任务目标
实现 workflow-board 的自动同步与归档能力，包括：
1. stage->board 自动同步（分钟级调度）
2. 事件归档机制
3. 跨归档分页查询

## 范围确认
- **包含**：自动同步调度、归档策略、跨层查询统一游标
- **不包含**：手动触发同步、实时推送、归档压缩优化
- **约束**：同步频率目标为分钟级（默认 1 分钟）

## 技术栈
- 调度：基于现有 Colony 调度基础设施
- 存储：复用现有 workflow 存储层
- 查询：统一游标协议（version/layer/event_id/ts）

## 成功标准
1. 同步延迟 p99 < 2 分钟
2. 归档查询性能不劣化（p95 增幅 < 10%）
3. 跨归档分页无重复/漏读
4. 幂等保证（重试不产生副作用）

## 风险识别
- 调度漂移可能导致同步延迟累积
- 归档层访问控制若弱于在线层会形成旁路风险
- 跨层时钟源不一致可能导致分页异常
