# Release Tag: Phase 6 Workflow Skill (v1.0.0)

## Overview
This release introduces the 8-stage development workflow skill for Colony agents, enabling structured collaboration with role-based checks and evidence validation.

## Features
- **State Machine**: 8 linear stages + rollback capability.
- **Roles**: Architect, Tech Lead, Developer, QA Lead.
- **Validation**: Mandatory evidence checks for stages > 0.
- **Audit Trail**: JSON-based history log.

## Artifacts
- Skill: `skills/dev-workflow/`
- Design: `skills/dev-workflow/SKILL_DESIGN.md`
- QA: `skills/dev-workflow/QA_CRITERIA.md`

## Sign-offs
- **Architect**: Architecture approved.
- **Developer**: Implementation complete.
- **QA Lead**: Integration tests passed.

## Deployment
Effective immediately for all new Colony tasks.
